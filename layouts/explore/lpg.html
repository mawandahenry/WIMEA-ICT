Low Power Gateway

Used to receive weather data from the nodes using the sink node and then send it to the WIMEA-ICT server via the internet using the electron modem

Description

The LPG was desingned by WIMEA-ICT project basing on the Particel elctron modem which is used to establish connections to the internet service Provider. The board provides a socket for insertion of the particle electron modem(click more details).
The LPG was designed to solve the problem of power shortages which faced the earlier Raspberry pi gateway. It was built for WIMEA-ICT Gen 3 Automatic weather staions.

Properties

  • DS3231 RTC clock module: For time stamping the weather data reports from the nodes
  • TP4056 Charging Protection circuit: RTC clock module: For charging the Lithium ion battery (LPG battery)
  • SD Card module: For connecting the SD card to the PCB
  • Particle Electron Modem: For establishing connection to the ISP so as the gateway can send data via internet Protocol
  • RSS2 Mote: Acts as the sink node to receive weather data from other nodes
